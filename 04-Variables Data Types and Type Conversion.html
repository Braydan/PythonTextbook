What Are Variables?

Think of a variable as a labeled box where you can store information. You give the box a name, and Python remembers what's inside so you can use it later! 

# Creating variables
name = "Alex"
age = 15
score = 95.5
is_student = True

print(name)        # Prints: Alex
print(age)         # Prints: 15
print(score)       # Prints: 95.5
print(is_student)  # Prints: True

Variable Naming Rules

  • Must start with a letter or underscore: name, _score ‚úì
  • Can contain letters, numbers, underscores: player1, high_score ‚úì
  • Cannot start with a number: 1player ‚ùå
  • Cannot contain spaces: my name ‚ùå (use my_name ‚úì)
  • Case-sensitive: score and Score are different!

üí° Good Naming Practice

Use descriptive names that tell you what's inside! player_health is much better than h. Your future self will thank you!

Data Types in Python

Python has different types of data, just like you have different types of things in real life. Here are the four main types you'll use most:

1. Strings (str) - Text

Strings are text - anything in quotes! Use them for names, messages, or any words. 

# Strings - always in quotes
username = "gamer123"
message = "Hello, world!"
favorite_game = 'Minecraft'  # Single or double quotes work!

print(type(username))  # Prints:

2. Integers (int) - Whole Numbers

Integers are whole numbers with no decimal points. Use them for counting, ages, scores, etc. 

# Integers - whole numbers, no quotes
age = 15
score = 1000
lives = 3
year = 2026

print(type(age))  # Prints:

3. Floats (float) - Decimal Numbers

Floats are numbers with decimal points. Use them for prices, measurements, percentages, etc. 

# Floats - numbers with decimals
price = 19.99
height = 1.75
temperature = 23.5
pi = 3.14159

print(type(price))  # Prints:

4. Booleans (bool) - True or False

Booleans can only be True or False. Use them for yes/no situations! 

# Booleans - True or False (no quotes!)
is_student = True
game_over = False
has_won = True

print(type(is_student))  # Prints:

‚úì Using type()

Use the type() function to check what type a variable is. Super useful when debugging!

Type Conversion

Sometimes you need to convert data from one type to another. Python makes this easy with built-in functions!

Converting to Integer: int()

# Converting to integer
text_number = "25"
real_number = int(text_number)
print(real_number + 5)  # Prints: 30

# From float to int (loses decimal!)
price = 19.99
whole_price = int(price)
print(whole_price)  # Prints: 19 (not 20!)

Converting to Float: float()

# Converting to float
text_price = "19.99"
real_price = float(text_price)
print(real_price * 2)  # Prints: 39.98

# From int to float
age = 15
age_float = float(age)
print(age_float)  # Prints: 15.0

Converting to String: str()

# Converting to string
age = 15
age_text = str(age)
print("I am " + age_text + " years old")  # Works!

# Can't add number to string without converting
score = 100
# print("Score: " + score)  # ERROR!
print("Score: " + str(score))  # Works!

‚ö†Ô∏è Watch Out!

int() cuts off decimals, it doesn't round!
int(9.9) gives you 9, not 10!

Why Type Conversion Matters

Example: User Input

Remember: input() always returns a string! You must convert it to do math. 

# Problem: Wrong types!
birth_year = input("What year were you born? ")
age = 2026 - birth_year  # ERROR! Can't subtract string

# Solution: Convert to int!
birth_year = input("What year were you born? ")
birth_year = int(birth_year)  # Convert to integer
age = 2026 - birth_year  # Now it works!
print("You are", age, "years old")

Example: Building Messages

# Problem: Can't add string and number
score = 95
# message = "Your score is " + score  # ERROR!

# Solution: Convert number to string!
score = 95
message = "Your score is " + str(score)
print(message)  # Prints: Your score is 95

# Or use commas in print (no conversion needed!)
print("Your score is", score)  # Also works!

üí° Quick Reference

int() ‚Üí Converts to whole number
float() ‚Üí Converts to decimal number
str() ‚Üí Converts to text
type() ‚Üí Tells you what type something is

Common Mistakes to Avoid

Mistake 1: Forgetting Quotes for Strings

# ‚ùå Wrong - no quotes
name = Alex  # ERROR: Python thinks Alex is a variable

# ‚úÖ Correct - with quotes
name = "Alex"  # This is a string!

Mistake 2: Quotes on Numbers You Want to Use in Math

# ‚ùå Wrong - this is a string, not a number!
age = "15"
next_year = age + 1  # ERROR! Can't add 1 to text

# ‚úÖ Correct - no quotes for numbers
age = 15
next_year = age + 1  # Works! Equals 16

Mistake 3: Mixing Types Without Converting

# ‚ùå Wrong - mixing string and number
score = 100
message = "Score: " + score  # ERROR!

# ‚úÖ Correct - convert to same type
score = 100
message = "Score: " + str(score)  # Works!

# ‚úÖ Or use commas in print
print("Score:", score)  # Works too!

Mistake 4: Forgetting to Convert Input

# ‚ùå Wrong - input is always a string!
num1 = input("Enter a number: ")
num2 = input("Enter another: ")
total = num1 + num2  # This joins text, doesn't add!

# ‚úÖ Correct - convert to numbers
num1 = int(input("Enter a number: "))
num2 = int(input("Enter another: "))
total = num1 + num2  # Now this adds numbers!

Mistake 5: Wrong Capitalization on Booleans

# ‚ùå Wrong - lowercase won't work
is_ready = true  # ERROR! Python doesn't know "true"
is_done = false  # ERROR! Python doesn't know "false"

# ‚úÖ Correct - capital T and F
is_ready = True
is_done = False

Key Takeaways

  1. Variables are labeled containers that store data
  2. Four main data types: strings (text), integers (whole numbers), floats (decimals), booleans (True/False)
  3. Use type() to check what type a variable is
  4. Strings need quotes, numbers don't, booleans are True/False (capital letters)
  5. int() converts to whole numbers (cuts off decimals)
  6. float() converts to decimal numbers
  7. str() converts to text
  8. input() always returns a string - convert it if you need a number!
  9. Can't mix types in operations without converting first
  10. Use descriptive variable names like player_score not x
