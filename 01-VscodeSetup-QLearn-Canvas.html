Lesson 01

Learn how to use Visual Studio Code (VSCode) to write and run Python programs on your lab computer.

What is VSCode?

Visual Studio Code (VSCode) is a powerful, free code editor made by Microsoft. It's one of the most popular tools that programmers use to write code in many different languages, including Python.

Think of VSCode as a super-powered text editor designed specifically for programming. It helps you write code by highlighting different parts in different colors, catching mistakes, and providing helpful suggestions as you type.

üí° Why VSCode?

VSCode is industry-standard software used by professional developers worldwide. Learning to use it now will give you skills that transfer to real-world programming jobs!

Opening VSCode on Your Lab PC

  1. Click the Start Menu (Windows logo in bottom-left corner)
  2. Type "Visual Studio Code" or look for the VSCode icon
  3. Click on Visual Studio Code to open the program
  4. Wait for VSCode to load - you should see the welcome screen

‚úì Success Check

You should see a dark or light interface with a sidebar on the left and a large editing area in the middle. If you see this, you're ready to start coding!

Creating Your First Python File

Step 1: Create a New File

  1. Click File ‚Üí New File (or press Ctrl + N)
  2. Click File ‚Üí Save As... (or press Ctrl + Shift + S)
  3. Choose a location to save your file (like Documents or Desktop)
  4. Name your file with a .py extension, for example: hello.py
  5. Click Save

üìÅ File Naming Rules

  • Always end Python files with .py
  • Use lowercase letters and underscores (e.g., my_program.py)
  • No spaces in file names (use underscores instead)
  • Make names descriptive (e.g., calculator.py instead of prog1.py)

Step 2: Write Your First Code

Type the following code into your new file:

print("Hello, Python!")

Step 3: Save Your File

Press Ctrl + S or click File ‚Üí Save

Running Your Python Code

Method 1: Using the Run Button (Easiest)

  1. Look for the ‚ñ∂ Run button in the top-right corner of VSCode
  2. Click it to run your code
  3. The output will appear in the Terminal panel at the bottom of the screen

Method 2: Using the Terminal

  1. Open the Terminal: View ‚Üí Terminal (or press Ctrl + `)
  2. Make sure you're in the right folder (where your Python file is saved)
  3. Type: python hello.py and press Enter
  4. You should see your output appear!

‚úì Expected Output

You should see: Hello, Python!

If you see this message, congratulations! You've just run your first Python program!

Essential VSCode Features for Python

1. Syntax Highlighting

VSCode automatically colors your code to make it easier to read. Keywords like print appear in one color, strings in another, and so on.

2. Auto-Complete

As you type, VSCode suggests code completions. Press Tab or Enter to accept a suggestion.

3. Error Detection

If you make a mistake, VSCode will underline it with a red squiggly line. Hover over the line to see what the problem is.

4. Indentation

Python uses indentation (spaces at the start of lines) to organize code. VSCode automatically indents your code, but you can also press Tab to indent manually.

‚å®Ô∏è Useful Keyboard Shortcuts

  • Ctrl + S - Save file
  • Ctrl + / - Comment/uncomment line
  • Ctrl + F - Find text
  • Ctrl + Z - Undo
  • Ctrl + Shift + P - Command palette

Common Problems and Solutions

Problem: "Python is not recognized"

Solution: Python might not be installed correctly. Ask your teacher to check that Python is installed on your lab computer.

Problem: Code doesn't run when you click Run

Solution: Make sure you've saved your file with a .py extension. The Run button only works with saved Python files.

Problem: Can't see the Terminal

Solution: Go to View ‚Üí Terminal or press Ctrl + ` (that's the backtick key, usually above Tab).

‚ö†Ô∏è Remember

Always save your work regularly (Ctrl + S)! It's a good habit that will prevent you from losing your code if something goes wrong.
