Lesson 02

Learn how to display information to users using the print() function.

What is Output?

Output is how your program communicates with the user by displaying information on the screen. In Python, the most common way to produce output is using the print() function.

Think of print() as your program's voice - it's how your code "speaks" to show results, messages, or any information you want the user to see.

üí° Why is Output Important?

Without output, your programs would run silently with no way to show results! Output lets you display answers, debug your code, and create interactive experiences for users.

The print() Function

Basic Usage

The simplest way to use print() is to display text (called a string) in quotes:

print("Hello, World!")

Output:

Hello, World!

üìù String Rules

  • Text strings must be enclosed in quotes: either "double quotes" or 'single quotes'
  • Both work the same way - just be consistent!
  • If you forget the quotes, Python will think it's a variable name

Printing Multiple Lines

You can use multiple print() statements to display several lines:

print("Welcome to Python!")
print("This is my first program")
print("Let's learn to code!")

Output:

Welcome to Python!
This is my first program
Let's learn to code!

Printing Different Types of Data

Numbers

You can print numbers without quotes:

print(42)
print(3.14159)
print(-17)

Output:

42
3.14159
-17

Mathematical Expressions

Python will calculate the result and print it:

print(5 + 3)
print(10 * 2)
print(100 / 4)

Output:

8
20
25.0

Printing Multiple Items

Separating with Commas

You can print multiple items in one print() statement by separating them with commas:

print("My name is", "Sarah")
print("I am", 14, "years old")
print("Score:", 95, "out of", 100)

Output:

My name is Sarah
I am 14 years old
Score: 95 out of 100

üí° Automatic Spacing

When you separate items with commas, Python automatically adds a space between them. This is very convenient for creating readable output!

Mixing Text and Numbers

You can mix strings and numbers in the same print() statement:

print("The answer is", 42)
print("Temperature:", 25, "degrees")
print("Price: $", 19.99)

Output:

The answer is 42
Temperature: 25 degrees
Price: $ 19.99

Changing the Separator and Ending

Custom Separator (sep)

By default, items are separated by a space. You can change this using the sep parameter:

print("Python", "is", "awesome")
print("Python", "is", "awesome", sep="-")
print("2025", "01", "24", sep="/")

Output:

Python is awesome
Python-is-awesome
2025/01/24

Custom Ending (end)

By default, print() adds a new line at the end. You can change this with the end parameter:

print("Hello", end=" ")
print("World!")

print("Loading", end="...")
print("Done!")

Output:

Hello World!
Loading...Done!

‚ö†Ô∏è Be Careful with end=""

If you use end="", the next print will continue on the same line. This is useful for creating custom output, but make sure it's what you want!

Escape Characters

Escape characters are special sequences that start with a backslash (\) and let you include special formatting in your strings.

Common Escape Characters

# New line
print("Line 1\nLine 2\nLine 3")

# Tab
print("Name:\tJohn")
print("Age:\t15")

# Quotes inside strings
print("She said, \"Hello!\"")
print('It\'s a beautiful day')

# Backslash itself
print("Path: C:\\Users\\Student")

Output:

Line 1
Line 2
Line 3
Name:	John
Age:	15
She said, "Hello!"
It's a beautiful day
Path: C:\Users\Student

üìã Escape Character Reference

  • \n - New line (moves to next line)
  • \t - Tab (adds horizontal spacing)
  • \" - Double quote character
  • \' - Single quote character
  • \\ - Backslash character

Printing Empty Lines

You can print blank lines to make your output more readable:

print("Welcome to my program")
print()  # Empty line
print("This is much easier to read")
print()  # Empty line
print("Because it has spacing!")

Output:

Welcome to my program

This is much easier to read

Because it has spacing!

Common Mistakes and How to Fix Them

Mistake 1: Forgetting Quotes Around Text

# ‚ùå Wrong
print(Hello)  # Python thinks Hello is a variable!

# ‚úÖ Correct
print("Hello")

Mistake 2: Mismatched Quotes

# ‚ùå Wrong
print("Hello World')  # Started with " but ended with '

# ‚úÖ Correct
print("Hello World")

Mistake 3: Forgetting Parentheses

# ‚ùå Wrong
print "Hello"  # Missing parentheses

# ‚úÖ Correct
print("Hello")

Mistake 4: Trying to Add Text and Numbers with +

# ‚ùå Wrong
print("Age: " + 14)  # Can't add string and number directly

# ‚úÖ Correct - Use a comma instead
print("Age:", 14)

# ‚úÖ Also Correct - Convert number to string
print("Age: " + str(14))

‚ö†Ô∏è Remember

When you see an error message about print, check these common mistakes first: quotes, parentheses, and mixing text with numbers incorrectly.

Key Takeaways

  • print() is used to display output to the screen
  • Text strings must be in quotes: "like this" or 'like this'
  • Numbers don't need quotes and can be used in calculations
  • Use commas to print multiple items with automatic spacing
  • Escape characters like \n and \t add special formatting
  • You can customize output with sep and end parameters
  • Empty print() creates blank lines for better readability
